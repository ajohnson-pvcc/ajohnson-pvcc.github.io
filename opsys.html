Operating Systems Terminology
Term Definition
operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
job queue a storage organization in which objects(in this case jobs) are ordered in first-in, first-out
FIFO first-in, first-out
interactive processing allows a program being executed remotely to carry on a dialogue with a user through remote terminals
time-sharing A feature that allows an OS to provide service to multiple users at the sime time
multiprogramming A technique which allows time-sharing by diving time into intervals and then the execution of each job is restricted to only one interval at a time. At the end of each interval, the current job is temporarily set aside and another is allowed to execute during the next interval
application software consists of the programs for performing tasks particular to the machine's utilization
Utlity software The majority of an installation's utility softeware consists of programs for performing activities that are fundamental to computer instalations but not included in the operating system. In a sense, utility software consists of software units that extend (or customize) the capabilities of the operation system
Graphical User Interface (GUI) Presents ojects to be manipulated, such as files and programs, pictorially on the display as visual elements such as icons
Shell Older user interfaces which communicated with users through textual messages using a keyboard and monitor screen
Kernel The internal part of an operating system which contains those software componets that perform the very basic functions required by the computer installation(such as the file manager)
Device Driver A componet of the kernel, software units that communicate with the controllers(sometime directly with peripheral devices) to carry out operations on the peripheral devices and translates generic requests into more technical steps required by the device assigned to that driver.
File Manager A part of the kernel, its job is to coordinate the use of the machine's mass storage facilities by maintaing records of all the files stored in mass storage including where it is located and where available unused portions are.
Memory Manager A part of the kernel, it is charged with the task of coordination the machine's use of main memory.
Virtual Memory "fictional" memory space created by using a technique called paging in which storage space on a magnetic disk is reserved for use as if it where memory storage space
Boot Strapping A procedure that is performed by a computer each time it is turned on. This procedure transfers the operating system from mass storage(where it is permanently stored) in to main memory(which is essentially empty when the machine is first turned on).
ROM Contains a collection of software routines for performing fundamental input/output activities such as receiving information from the keyboard, displaying messages on the screenm and reading data from mass storage. Stored in nonvolatile memory such as FlashROM, this software is a permanent part of hardware, but is also not as readily changeable as the other data stored in mass storage(software).These collections AKA Firmware such as BIOS, EFI, and CFE.
deadlock a condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
spooling A technique of holding data for output at a later more convient time
super user (or adiministrator) Highky privileged access to the operating system granted during the login procedure. Once established, the adiministrator can alter settings within the operating system, modify critical software packages, adjust privileges granted to other uses, and perform a variety of other maintenance activities that are denied normal users
multi-factor authentication A solution to the pervasive problem of users using weak password to secure their computer accounts is to require more than one piece of evidence to gain access. Additional authentications such as an ATM card, specialized security tokens,or smartphone with a particular number are used in conjuction with a password to valid user authenticity.
