Network Classifications
Term Definitions
Network Classifications
    LAN: local area network - Links computers within a single building or complex
    MAN: Metropolitan area network - links machines in a network of intermediate size such as a town or city
    WAN: Wide area network - links machines over great distances such as between cities or countries
Network Models
    Bus Network - All machines are connected to a central controller or bus
    Star Network - A central machine connected to other machines which serve as an access point to the central machine
Protocols
    CSMA/CD Ethernet(Carrier Sense, Multiple Access with Collision Detection) - Dictates that each message be broadcast to all machines on the bus. . Each machine monitors all the messages but keeps only those addressed to itself. To transmit a message, a machine waits until the bus is silent, and at this time it begins transmitting while continuing to monitor the bus. If another machine also begins transmitting, both machines detect the clash and pause for a brief, independently random period of time before trying to transmit again. The result is a system similar to that used by a small group of people in a conversation. If two people start to talk at once, they both stop. T
    CSMA/CA Wifi - The protocol used is similar to Ethernet’s CSMA/CD. The basic difference is that when a machine first needs to transmit a message and finds the communication channel si1lent, it does not start transmitting immediately. Instead, it waits for a short period of time and then starts transmitting only if the channel has remained silent throughout that period. If a busy channel is experienced during this process, the machine waits for a randomly determined period before trying again. Once this period is exhausted, the machine is allowed to claim a silent channel without hesitation. This means that collisions between “newcomers” and those that have already been waiting are avoided because a “newcomer” is not allowed to claim a silent channel until any machine that has been waiting is given the opportunity to start
Combining networks
    repeater - a device that passes signals back and forth between the two original buses without considering the meaning of the signals
    bridge - Connects two buses, but doesnt necessarily pass all messages across the connection. Instead, it looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side. Thus, two machines residing on the same side of a bridge can exchange messages without interfering with communication taking place on the other side.
    switch - a bridge with multiple connections, allowing it to connect several buses rather that just two. Considers the destination addresses of all messages and forwards only those messages destined for other spokes. Each message that is forwarded is relayed only into the appropriate spokem thus minimizing the traffic in each spoke.
    router - Handles the connection between networks to form an internet, whose special purpose is to forward message. Provides links between networks while allowing each network to maintain its unique internal characteristics
    gateway - The "point" at which one network is linked to an internet. Serves as the passageway between the network and the outside work
Client/server model
    P2P: peer-to-peer - involves processes that provide service to and recieve service from each other
    Distributed System - consist of software units that execute as processes on different computers
    Cloud Computing - huge pools of shared computers on a network can be allocated for use by clients as needed.
Internet addressing
    Tier 1 - ISPs that consist of very high-speed, high capacity, internation WANs
    Tier 2 - ISPs that tend to be more regional in scope and less potent in their capabilties. Tend to be operated by companies in the communications business
    Tier 3 - ISPs - AKA access ISP - An independent Internet (Intranet) operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.

Internet four-layer software
    application layer - Prepares message and provides destination address/Recieves Message
    transport layer - Chops message into packets / Collects packets and reassembles message
    network layer - Assigns intermediate address to each packet/ Detects that packet has reached its final destination
    link layer - Transfer packet / Recieves Packet

Cybersecurity
    malware -Collective name for malicious Software
    virus - Software that infects a computer by inserting itself into programs that already reside in the machine.
    worm - an autonomous programs that transfer itself through a network, taking up residence in computers and forwarding copies of iteself to other computers.
    Trojan horse - a program that enters a computer system diguised as a desirable program, such as a game or utility package. Once installed it performs additional activities that might have harmful effects.
    Spyware - software that collects information about acitivies at the computer in which it resides and reports that information about activies back to the instigator of the attack.
    Phishing - a technique of obtaining information explicity by simply asking for it
    DDoS attack (dedicated denial of service attack) - the process of overloading a computer with messages or requests.
Protection
    firewall - A primary prevention technique used to filter traffic passing through a point in the network
    spam filter - A filter designed specifically to block unwanted email
    proxy server - a preventative tool that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of a server.(Prevents a server from directly have access to client information)
    antivirus software - Used to detect and remove the presence of known virus and other malware
    Cryptography - the study of sending and receiving messages securely in the presence of adversaries
    Public Key Encryption - the encoding of information in an effort to keep it confidential
